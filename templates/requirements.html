Big Idea / Goal / Why did we do this?

The main objective of our project is to provide a simple and straightforward website that can provide sentiment analysis for any keyword users put in. Sentiment analysis is a process of analyzing the emotional tone behind the series of words. Through sentiment analysis, users can identify if the keyword is positive, neutral, or negative. Moreover, it is one of the most commonly used tools for analyzing opinions about a specific topic. We wanted to create an accessible and easy-to-use website that does the hard work for numerous companies that conduct sentiment analysis. Rather than having to create a whole programming application to conduct sentiment analysis, companies can simply go on our website to get sentiment-related information about any topics. The sentiment analysis would be based on tweets posted on the Twitter server. Since Twitter is one of the most popular social media platforms, nowadays, we believe that Twitter would provide a holistic, comprehensive view of the overall sentiment of the public. We will provide three specific analyses, which includes polarity, subjectivity, and the overall sentiment. Our minimum viable product would be a website that provides the basic data regarding sentiment analysis such as polarity and subjectivity. For our stretch goal, we plan to provide visualized data analysis of the sentiment, which would include pie charts and scatter plots that highlight historical changes in the sentiment of a specific keyword. At that point, we may have to consider creating a business and charging political campaigns and think tanks to access our technology.

User Instruction / READ ME

If you want to get the general sentiment about a specific keyword on Twitter, you can use our Overall Sentiment Analysis option. This also allows you to compare the general sentiment of two different keywords. You can utilize these tools to compare the sentiment of two rivalry keywords (e.g. Trump vs Obama, Lakers vs Celtics, etc.) that you would like to analyze. Moreover, it also provides a comparative bar chart of the two keywords, which would allow you to better identify the differences among the sentiments of two keywords.

If you prefer to get the sentiment of a specific Twitter account regarding various topics, you can utilize our Individual Sentiment Analysis option above. This option allows you to input as many keywords as you would like to analyze. For example, you can analyze the sentiments of Donald Trump's tweets about the impeachment inquiry. You must put in a specific Twitter username (e.g. @elonmusk), for us to analyze the sentiments! It also provides a several informative charts that depicts the historical changes in the length of the tweets and historical changes of the likes and retweets of the tweets regarding the selected topics posted by the selected twitter account.

Implementation Information

We first created a python code that allows users to input the keywords that they wish to analyze in the visual studio. Once we created a code that provides sentiment analysis and visualizations, we decided to display our sentiment analysis software in the form of website. We wanted our software to work on a website to allow users to use our software more easily. In order to display our python codes into the form of website, we created a website using Flask and Heroku. Through the use of our codes, users simply can access our website using the url link. There, they can simply type in the keywords they wish to analyze and receive sentiment analysis.
